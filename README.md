# Tarea-S13.Between_Ghosts_and_Graphs: A Data-Driven reading of The Canterville Ghost.

### Análisis literario de The Canterville Ghost mediante análisis de datos

### Descripción del proyecto

Este repositorio contiene un proyecto de análisis literario aplicado al cuento The Canterville Ghost de Oscar Wilde (1887), utilizando herramientas y metodologías propias del análisis de datos. El objetivo principal es explorar hasta qué punto un texto literario puede ser analizado mediante técnicas computacionales sin perder su complejidad interpretativa.

El estudio se centra en:

- Análisis estilístico: estructura del texto, longitud de frases, ritmo narrativo.

- Análisis temático: identificación de campos semánticos (gótico, cómico, cultural y neutro).

- Análisis afectivo: aproximación al tono emocional del texto mediante polaridad léxica.

### Metodología

Procesamiento del texto y limpieza manual del corpus.

Construcción de un DataFrame:

- Orientado al análisis del estilo y la estructura.

- Otro basado en escenas para el análisis temático y afectivo.

- Uso de visualizaciones exploratorias (nube de palabras, gráficos de barras, boxplots, líneas, heatmaps y grafos).

- Interpretación de los resultados desde una perspectiva literaria.

### Herramientas utilizadas

Python
pandas
matplotlib

### Resultados

Los resultados muestran que Oscar Wilde articula el relato mediante una alternancia estilística consciente, una superposición de registros temáticos y una distancia afectiva marcada por la neutralidad léxica. El análisis relacional de personajes refuerza la idea de que el núcleo del cuento reside en las interacciones más que en el elemento sobrenatural en sí.

### Conclusiones

El proyecto demuestra que las metodologías de análisis de datos pueden aplicarse al estudio literario como herramientas complementarias a la lectura tradicional. Lejos de reducir el texto a cifras, este enfoque permite identificar patrones, ritmos y estructuras que enriquecen la interpretación crítica.
